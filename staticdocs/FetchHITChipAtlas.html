FetchHITChipAtlas

Usage

FetchHITChipAtlas(allowed.projects, dbuser, dbpwd, dbname, result.path, my.scaling = "minmax",
  mc.cores = 3, remove.nonspecific.oligos = FALSE)

Arguments

allowed.projects
Vector of projectNames to include
dbuser
MySQL username
dbpwd
MySQL password
dbname
MySQL database name
result.path
Directory for storing the results.
my.scaling
Normalization method. Default "minmax".
mc.cores
Number of cores for parallel computation
remove.nonspecific.oligos
Optional logical set to remove nonspecific oligos (TRUE). By default keeping the non-specific oligos (FALSE) Returns:

Value

List with the following elements: training.data: training data matrices (including alternative preprocessing methods) test.data: test data matrices full.data: full data matrices atlas.sampleinfo: sample information matrix parameters: input parameters and R session information

Description

Description: Complete preprocessing pipeline for the HITChip atlas with default parameters
