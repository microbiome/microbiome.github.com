read.profiling

Usage

read.profiling(level = NULL, method = "rpa", data.dir = NULL, log10 = TRUE, impute = TRUE)

Arguments

level
phylogenetic level ("oligo" / "species" / "L1" / "L2" / "L0") or "phylogeny.info"
method
("rpa" / "sum" / "ave" / "nmf")
data.dir
Profiling script output directory for reading the data. If not given, GUI will ask to specify the file and overruns the possible level / method arguments in the function call.
log10
Logical. Logarithmize the data TRUE/FALSE. By default, the data is in original non-log scale. Returns:

Value

data matrix (phylo x samples)
