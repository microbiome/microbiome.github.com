Description: Calculate Wilcoxon test (unpaired; BH correction) for the specified sample groups. Either provide the input data as matrix, file path, or select the file through GUI.

Usage

check.wilcoxon(dat = NULL, fnam = NULL, p.adjust.method = "BH", sort = FALSE)

Arguments

dat
data matrix (features x samples)
file
data file (if data matrix not provided)
p.adjust.method
p-value correction method for p.adjust function (default "BH")
sort
sort the results Returns:

Value

(Corrected) p-values for two-group comparison.
